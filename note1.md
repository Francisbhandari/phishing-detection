Phishing - An attempt to steal sensitive information like username, password, bank details or other important data to sell or utilize them.

URL - Uniform Resource Locator, a link or web address used to locate a specific site in the Internet.

URL Phishing - A cyber attack where malicious links or phishing sites are sent to redirect into fake websites to steal sensitive information.

Some of the patterns in URL phishing:
1. Misspellings
Attackers mostly use domain name which are misspelled, like missing letters, swapped letters, extra letters, etc. because people tend to not read the name letter by letter. Like gogle.com, microsft.com, etc

2. Look Alike Characters
Some letters/characters look alike, like '0 and O', 'l and 1', etc. Our eyes can see the shape, not the exact letters.

3. Extra words that sounds legit
Attackers put some trustworthy sounding words like secure, verify, login, update, etc. in the domain name to sound legit.

4. Too many sub-domains
Some links have many layers before the real domain to make it confusing. 

5. Unusual Domain Ending
Some websites have unusual endings like .top, .xyz, .click, etc. which is mostly ignored by people.

6. Too many hyphens
Suspicious URLs contains too many hyphens to copy real brand names.

7. Strange numbers in domain
Random numbers are added in domain name to make it look like real Id and official.

8. IP address in domain name
Some links use IP address's numbers in the domain name because most people don't recognize IP address.

9. Port numbers
Sometimes the domain name can contain unusual port numbers after a colon, like :8088, etc. which looks technical so people ignore it.

10. Very long URLs
Some URLs are made very long to hide the dangerous parts, because people tend to look only at the beginning part.

11. "@" symbol
Attackers use @ symbol to hide the real domain. Anything before @ is treated as username info. Example: "google.com@mysite.xyz", here "google.com" is not the real destination, "mysite.xyz" is.

12. Too many special characters
Some URLs may contain too many special characters like hyphen and underscore.

13. Urgency message
Some links contains scary or urgent messages like "Your account is locked. Verify now quickly" which makes the user act quickly without checking.
